Here is the **summary table** of all the Moodle plugins we've developed or designed together, along with their purpose:

| Plugin Folder Name           | Purpose                                                                 |
| ---------------------------- | ----------------------------------------------------------------------- |
| `local/tekouin_apibridge`    | Exposes Moodle data via custom REST APIs for external dashboards        |
| `local/oneclickexport`       | Adds one-click course export to `.mbz` format                           |
| `local/tekouin_gradegen`     | Automatically generates gradebook structure with weighted sections      |
| `local/tekouin_labs`         | Integrates external Cybersecurity, Cloud, and Big Data labs into Moodle |
| `local/tekouin_vmsim`        | Provides Windows VM and 3D lab simulation access for learners           |
| `local/tekouin_gamification` | Connects Moodle with Tekouin gaming platform for gamified learning      |
| `mod/gameengine`             | Activity module to embed educational games within courses               |
| `local/tekouin_teacherhub`   | Links Moodle with external teacher analytics and discovery platform     |
| `local/tekouin_mtnpay`       | Enables MTN Mobile Money payments for course enrollment or services     |
