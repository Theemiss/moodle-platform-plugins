# ✨ ChatGPT_Assistant_Front  
### 🌐 For Tekouin Learning Platform  

## 🚀 Project Overview  

The **ChatGPT Assistant** plugin for Moodle enhances the **Virtual Programming Lab (VPL)** by integrating the **OpenAI API** to generate **hints** for students' programming exercises. Instead of real-time chat, students click a button to receive AI-generated hints and step-by-step guidance based on their code submissions, simulating a conversational experience while encouraging independent problem-solving.
### 🌟 Key Benefits  

- 🎓 **Effective Learning**  
  The assistant encourages active engagement by offering tailored hints and solutions, helping students reinforce their coding and problem-solving skills.  

- 📈 **Increased Retention on Platform**  
  By addressing students' needs efficiently within Tekouin, the assistant reduces the need to seek external resources, ensuring students stay on the platform.  

- 🛡️ **Minimized Cheating**  
  By guiding students through problems rather than providing direct answers, the assistant fosters genuine learning and academic integrity.

## 🔧 How to Use

1. **Download the ZIP file**  
   Obtain the plugin's ZIP file from github release and save it to your local system.  

2. **Set up Moodle and LAMP stack**  
   Ensure Moodle is installed and a LAMP (Linux, Apache, MySQL, PHP) stack is properly configured on your local machine.  

3. **Access Moodle on localhost**  
   Render Moodle by navigating to `http://localhost` in your web browser to verify the installation is running.
